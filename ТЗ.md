# Техническое задание (ТЗ)
## Проект: Telegram AI-бот для записи клиентов с онлайн-оплатой

---

## 1. ЦЕЛЬ ПРОЕКТА

Разработать Telegram-бота, который:
- общается с клиентом с помощью AI
- записывает клиента на услугу
- предлагает оплату
- уведомляет администратора

---

## 2. АРХИТЕКТУРА

Компоненты:

1. Telegram Bot (aiogram)
2. Backend API (FastAPI)
3. База данных (SQLite/PostgreSQL)
4. AI API (DeepSeek / GigaChat / OpenRouter)
5. Платежный сервис (ЮKassa / CloudPayments)

Связь:


Telegram → Bot → Backend → AI / DB / Payment


---

## 3. ФУНКЦИОНАЛ

### 3.1 Telegram-бот

Команды:

- /start — запуск
- /help — помощь

Основные сценарии:

- клиент пишет сообщение
- бот понимает запрос (через AI)
- бот ведёт диалог
- собирает данные:
  - услуга
  - дата
  - время
  - имя
  - телефон

---

### 3.2 AI-логика

AI должен:

- определять intent:
  - booking (запись)
  - question (вопрос)
- извлекать данные:

```json
{
  "intent": "booking",
  "service": "стрижка",
  "date": "завтра",
  "time": "15:00",
  "name": null,
  "phone": null
}
```
указывать недостающие поля
3.3 Управление записью

Бот:

задаёт уточняющие вопросы
проверяет заполненность данных
предлагает варианты времени (если нет)

После сбора:

создаёт запись в БД
3.4 Онлайн-оплата

После создания записи:

бот отправляет:
сумму
описание услуги
кнопку оплаты

После оплаты:

статус записи → "оплачено"
отправляется подтверждение клиенту
3.5 Уведомления

Бот отправляет:

Клиенту:

подтверждение записи
подтверждение оплаты

Администратору:

новая запись
информация:
имя
услуга
дата/время
телефон
3.6 История

Клиент может:

посмотреть свои записи
отменить запись
