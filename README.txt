UserService
- создать пользователя при /start
- найти пользователя по telegram_id
- обновить имя/телефон

CatalogService
- получить список услуг
- найти услугу по id/name
- получить цену/длительность

BookingService
- создать запись
- проверить свободный слот
- получить мои записи
- отменить запись
- изменить статус pending/paid/cancelled
